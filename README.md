# Mah-Jong Tracker 
Keep track of Mah-Jong points. 

## Installation 
1. `git clone https://github.com/jasonkwok/Mah-Jong-Tracker.git` 
2. `g++ mahjong-tracker.cpp -o mahjong-tracker` 

## Usage 
1. `./mahjong-tracker` 

## Point System Used 
Faan points | Base points
--- | ---
1 | 2
2 | 4
3 | 8
4 | 16

8 | 96
